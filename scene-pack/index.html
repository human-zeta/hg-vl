Cómo usar el pack en OBS
Un solo archivo sirve las 4 escenas según ?scene=. En OBS creás una escena por cada una (Starting / BRB / Ending / Technical) y en cada una agregás un Navegador → Archivo local apuntando al mismo scene-pack.html, y le agregás el parámetro en el campo URL local — o usás estos valores:
  1. Starting Soon → scene-pack.html?scene=starting
  2. Ya vuelvo / BRB → scene-pack.html?scene=brb
  3. Cerrando → scene-pack.html?scene=ending
  4. Dificultades técnicas → scene-pack.html?scene=technical
Ancho / Alto = el formato elegido. Si tu OBS no deja pasar ?scene= en archivo local, descargá también los PNG o subí el HTML a un hosting y usá la URL con el parámetro.
Tip: el countdown arranca al mostrar la fuente. Para reiniciarlo antes de salir en vivo, clic derecho en la fuente → Actualizar caché.
