Cómo dispararla en OBS / Stream Deck
El HTML reproduce la alerta al cargar la página. Como una Browser Source carga una vez, para re-dispararla en vivo tenés dos caminos:
  1. Stream Deck / hotkey → acción "Refresh Browser Source". Cada vez que la apretás, la alerta suena y se anima de nuevo.
  2. Escena dedicada con la fuente y la opción "Actualizar el navegador cuando la escena se active"; entrás y salís de la escena para dispararla.
Nombre y monto dinámicos: pasás datos por la URL de la fuente. Tu URL local (ejemplo con lo que tenés ahora):
alerta.html?name=JUANCITO&amount=$5&msg=¡grande!
Marcá "Apagar la fuente cuando no está visible" + "Actualizar al activarse" para que el sonido no se adelante. El PNG es el cuadro fijo de la alerta (sin sonido). Para alertas automáticas conectadas a Twitch/YouTube hace falta un backend — esto es la capa visual, disparable a mano o por Stream Deck.
